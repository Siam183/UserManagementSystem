UserManagementSystem

A web-based User Management System built with ASP.NET Core MVC, Entity Framework Core, SQL Server, and Cookie Authentication.

Features
User Registration and Login
Secure Password Hashing
Email Verification via Gmail SMTP
Cookie-Based Authentication
User Status Management (Unverified, Active, Blocked)
Block and Unblock Users
Delete Users
Delete Unverified Users
Bulk User Actions with Multi-Select Support
Automatic Logout for Blocked or Deleted Users
Last Login Tracking
Responsive Bootstrap User Interface
SQL Server Database Integration
Entity Framework Core Migrations
Technologies Used
ASP.NET Core MVC (.NET 8)
Entity Framework Core
SQL Server
Cookie Authentication
Bootstrap 5
Gmail SMTP
C#
Project Overview

This application provides a complete user account management workflow, including registration, authentication, email verification, and administrative user management. The system supports bulk operations, status tracking, and secure access control while maintaining a responsive and user-friendly interface.

Key Security Features
Passwords stored using secure hashing
Authentication using ASP.NET Core Cookies
Email verification before account activation
Automatic session invalidation for blocked or deleted users
Unique email enforcement through database constraints
